🔄 Auto-Sync with Gumroad

Products currently load from static data. To auto-sync whenever you add a product on Gumroad, integrate the Gumroad API: GET https://api.gumroad.com/v2/products. The data structure matches the API format — swap the static array for a fetch() call. See sync-gumroad.js for the Node.js migration script.
